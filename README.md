# scrapy_private

本框架使用方法和scrapy基本一致,是模仿scrapy,自定义了部分的功能

## 已经实现的功能如下:
- 支持分布式(使用分布式需要安装redis, 用户可自定义是使用单机版还是分布式)
- 支持断点续爬, 避免重复工作
- 支持增量式爬虫的爬取
- 支持了重复的请求过滤, 用户可自行设置使用
